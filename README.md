Notes app by Manuel Menendez Balbi
Can run the app either by running docker-compose up in the main directory (recommended) or running the sh script.

Tools used:
	- Java OpenJDK 17
	- Spring Boot 3.2.2 (Spring Web, Spring Data JPA)
	- Gradle 8.2.1
	- MySQL latest
	- React 18.2.0
	- npm 8.11.0
	- node 18.16
